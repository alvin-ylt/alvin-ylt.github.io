How I began?

When I was in secondary school, I began my journey to explore programming and networking technology with hobby projects (including this website!).

The enhancement of web security with Cloudflare proxy, DNSSEC and HSTS especially intrigued me to learn about cyber security and its underlying mathematical principles.

I started learning to create programs with Python, as it's renowned as an "intuitive" programming language for beginners. Though I wasn't aware of program efficiency, it helped me broaden my horizons to algorithms. Some of my first programs are showcased here. Later on, I also attempted to create a Telegram chatbot with its API and hosted it with Raspberry Pi.

During the internship at Aura Labs, I also had the first glance of Node.js as I created tools to convert documentation formats with JavaScript. Meanwhile, I developed the concept of time complexity, DevOps deployment and frontend/backend development.

In fact, I did not learn programming formally until the first semester at university, where we work on more extensive projects with Haskell. One of the projects was to develop software agents to play a board game. The declarative programming language resembles mathematical functions and is much more type-sensitive than Python. This brought my attention to type theory and its relationship with high-level programming languages.
